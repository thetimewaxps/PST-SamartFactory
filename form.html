ฝ่ายขาย — วิธีใช้งาน

📋 BreakDown
ใส่ข้อมูลวัตถุดิบ (น้ำหนัก, ราคา/กก.), ค่าแรง, ค่าใช้จ่ายต่างๆ เพื่อคำนวณต้นทุนและราคาเสนอลูกค้า ใช้ร่วมกับแท็บแรงงานและ MAT เพื่อดึงข้อมูลประกอบ
📝 ใบเสนอราคา / Order
บันทึกงานจากลูกค้า (PO, ชนิดสินค้า, จำนวน, ราคา) ติดตาม Process/สถานะงาน ดูประวัติ แก้ไข และยกเลิก Order ได้ในหน้านี้
🗂️ DATA
จัดการฐานข้อมูลสินค้า (ชื่อ, OD, WT, Grade, ราคาขายอ้างอิง) ใช้เป็นต้นแบบดึงข้อมูลเข้า BreakDown อัตโนมัติ
👷 แรงงาน
ตั้งค่าค่าจ้างแรงงาน (บาท/ลูก หรือ บาท/กก.) ระบบดึงข้อมูลนี้ไปคำนวณต้นทุนแรงงานใน BreakDown อัตโนมัติ
🔩 แม่พิมพ์
บันทึกข้อมูลแม่พิมพ์ของลูกค้า (เลขแม่พิมพ์, สภาพ, ที่เก็บ) ติดตามการใช้งานและค่าเสื่อม
🧱 MAT
จัดการข้อมูลวัตถุดิบ (ราคา/กก., Spec) ใช้เป็นราคากลางในการคำนวณต้นทุน BreakDown
⚡

เงื่อนไขที่กำหนดในระบบ

🔄ออก ใบกำกับภาษี → Process ของ Order ที่เกี่ยวข้องเปลี่ยนเป็น เตรียมส่ง อัตโนมัติ (ยกเว้นสถานะเป็น "เรียบร้อย" แล้ว)
🔄ออก ใบส่งชุบ → Process เปลี่ยนเป็น ส่งชุป อัตโนมัติ (ยกเว้นสถานะเป็น "เตรียมส่ง" หรือ "เรียบร้อย" แล้ว)
❓ยกเลิก ใบกำกับ หรือ ใบส่งชุบ → ระบบจะถามว่าต้องการเปลี่ยน Process กลับเป็นอะไร ก่อนทำการยกเลิก
📊ลำดับ Process: รอConfirm → กำลังผลิต → ส่งชุป → ส่งตัวอย่างเทส+รอสรุป → เตรียมส่ง → ส่งยังไม่ครบ → FG รอเรียก → Stock → เรียบร้อย
🛒

ฝ่ายจัดซื้อ — วิธีใช้งาน

🧾 ใบสั่งซื้อ (PO)
สร้างและติดตามใบสั่งซื้อวัตถุดิบจาก Supplier บันทึกวันสั่ง, วันรับ, ราคา, จำนวน และสถานะการรับของ
🧪 ใบส่งชุบ
ออกใบนำส่งชิ้นงานไปชุบ ระบุ Order ที่เกี่ยวข้อง, ร้านชุบ, จำนวน เมื่อบันทึกจะอัปเดต Process ของ Order อัตโนมัติ
● คอลัมน์ "จำนวน" แก้ไขได้ (ช่องขอบเหลือง) — กรอกจำนวนที่จะส่งชุบจริง ไม่กระทบยอด Order; เปลี่ยนค่าจะอัปเดตจำนวน ฝาบน/ฝาล่าง/ตะแกรง ให้อัตโนมัติ
● คอลัมน์ "ตะแกรงกลาง" — ค่าเริ่มต้น ❌ ทุกรายการ (ไม่ผูกกับ Order); เปิดได้เฉพาะในรายการเพิ่มเอง (Extra items)
● ประวัติใบส่งชุบ: ปุ่ม ✏️ แก้ไข — แก้วันที่, ร้านชุบ, จำนวน และราคา/หน่วย ของแต่ละรายการได้ (ไม่ต้องลบแล้วออกใหม่)
🧱 MAT
ดูและอัปเดตราคาวัตถุดิบ ใช้เป็นราคากลางในการสั่งซื้อและตรวจสอบต้นทุน
🏢 Supplier
จัดการข้อมูล Supplier/ผู้ขาย (ชื่อ, เบอร์, ประเภทสินค้า) ใช้อ้างอิงในใบสั่งซื้อและใบส่งชุบ

❓ยกเลิก ใบส่งชุบ → ระบบถามว่าต้องการเปลี่ยน Process กลับเป็นอะไร ก่อนทำการลบ
⚠️หาก Process ของ Order เป็น "เตรียมส่ง" หรือ "เรียบร้อย" อยู่แล้ว การออกใบส่งชุบจะ ไม่เปลี่ยน Process (ป้องกันการถอยหลัง)

ฝ่ายผลิต — วิธีใช้งาน

📋 Job Order
ดูรายการงานที่ต้องผลิต กรองตาม Process/สถานะ เปลี่ยน Process ของงานได้โดยตรง คลิก Order เพื่อดูรายละเอียดครบถ้วนพร้อม Timeline และรูปภาพ
🔢 คำนวณตัดเหล็ก
คำนวณการตัดเหล็กแผ่น/เหล็กแท่ง ใส่ขนาดและจำนวนเพื่อหาจำนวนชิ้นและของเหลือจากแต่ละแผ่น
🔩 แม่พิมพ์
ดูและจัดการข้อมูลแม่พิมพ์ ตรวจสอบสภาพ ที่เก็บ และประวัติการใช้งาน
🚀 ติดตามงาน
มุมมองสรุปสถานะงานแบบภาพรวม (แสดงเฉพาะส่วนที่เกี่ยวกับการผลิต) ใช้กรองและติดตาม Order ตาม Process
🔍 Inspection / 📄 WI
⏳ กำลังพัฒนา ยังไม่พร้อมใช้งาน

เงื่อนไขที่กำหนดในระบบ

✏️แผนกผลิตสามารถ เปลี่ยน Process ของ Order ได้เองผ่าน Job Order (กดปุ่ม 3 จุด หรือคลิกเข้าดูรายละเอียด)
🔄Process บางอย่างถูกเปลี่ยน อัตโนมัติจากแผนกอื่น: ฝ่ายจัดซื้อออกใบส่งชุบ → ส่งชุป, บัญชีออกใบกำกับ → เตรียมส่ง
📊ลำดับ Process ที่แนะนำ: รอConfirm → กำลังผลิต → ส่งชุป → เตรียมส่ง → เรียบร้อย (สามารถข้ามได้ตามความเหมาะสม)
💰

บัญชี — วิธีใช้งาน

📑 ใบกำกับภาษี
ออกใบกำกับภาษีจาก Order เลือก Order ที่ต้องการออกบิล ระบุวันที่ ระบบสร้างเลขใบกำกับให้อัตโนมัติ และอัปเดต Process ของ Order ที่เกี่ยวข้อง
👥 เพิ่มลูกค้า
จัดการข้อมูลลูกค้า (ชื่อ, ที่อยู่, เลขประจำตัวผู้เสียภาษี) ข้อมูลจะถูกดึงไปใช้ในใบกำกับภาษีและใบเสนอราคา
📊 รายงานภาษีขาย
ดูรายงานสรุปยอดขายและภาษีมูลค่าเพิ่มตามช่วงเวลา ใช้สำหรับยื่นแบบ ภ.พ.30 รายเดือน
📑 ใบวางบิล
สร้างใบวางบิลรวมใบกำกับหลายใบให้กับลูกค้า ระบุเงื่อนไขชำระเงิน วันครบกำหนด พิมพ์หรือส่งให้ลูกค้าได้
